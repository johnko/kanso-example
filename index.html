Simplest possible app

This example app is from http://kan.so/docs/Simplest_possible_app

  1. Get the dependencies before pushing:

    kanso install

  2. Push the app to a couchdb database:

    kanso push http://localhost:5984/example

  3. Upload the initial data documents:

    kanso transform add-ids data/example.json data/example-with-ids.json
    kanso upload data/example-with-ids.json http://localhost:5984/example

  4. The app will be deployed to:
    http://localhost:5984/example/_design/example-app/index.html
